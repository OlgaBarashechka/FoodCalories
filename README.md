# FoodCalories
FoodCalories
Нейросеть для оценки калорийности блюд: постановка задачи
Цель проекта
Разработать и обучить глубокую нейронную сеть (DL‑модель), которая по двум типам входных данных — фотографии блюда и списку ингредиентов — прогнозирует его калорийность.

Область применения
Готовая модель может быть интегрирована в фитнес‑приложения и сервисы о здоровом питании. Решение будет полезно пользователям, которые контролируют рацион и следят за энергетической ценностью потребляемой пищи.

Входные данные
Для обучения и валидации модели предоставляется датасет, содержащий:

фотографии тарелок с различными блюдами;
текстовые описания ингредиентного состава каждого блюда.

Ожидаемый результат
Сформированная DL‑модель, способная на основе изображения и списка ингредиентов выдавать оценку калорийности блюда (в ккал).

Критерии готовности модели к внедрению
Модель считается пригодной для интеграции в продуктивную среду, если на тестовой выборке достигает значения метрики MAE (Mean Absolute Error) менее 50
